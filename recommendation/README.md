# Recommendation Pipeline

Produces a credit score using the Five Cs of Credit, derives a loan decision (amount, rate, tenor), and generates a professional Credit Appraisal Memo (CAM) as Word (.docx) and PDF.

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/recommend/` | Score + decide + generate CAM |
| `GET` | `/recommend/cam/{filename}` | Download CAM as .docx |
| `GET` | `/recommend/pdf/{filename}` | Download CAM as .pdf |

Pass the full output from `/research/analyze` along with loan-specific fields:

## Five Cs Scoring Engine

All scoring is **rule-based and transparent** — every point deducted or awarded includes an explanation.

### Character (max 25)
- Deductions for risk signals by severity and type (fraud/litigation/regulatory carry extra penalty)
- Deductions for promoter-level risks
- Full score if no adverse signals found

### Capacity (max 25)
- **DSCR** (10 pts): ≥1.5 = 10, ≥1.25 = 8, ≥1.0 = 5, <1.0 = 2
- **Profit margin** (8 pts): ≥15% = 8, ≥8% = 6, ≥0% = 4, loss = 0
- **Cashflow volatility** (7 pts): low = 7, moderate = 4, high = 1

### Capital (max 20)
- **D/E ratio** (10 pts): <1 = 10, <2 = 7, <3 = 4, ≥3 = 1
- **Net worth coverage** (10 pts): ≥3x loan = 10, ≥2x = 7, ≥1x = 4, <1x = 1

### Collateral (max 15)
- LTV ≤50% = 15, ≤70% = 11, ≤80% = 7, >80% = 3
- Unsecured = 5

### Conditions (max 15)
- Baseline 10, sector trends shift ±5 based on keyword sentiment analysis
- Industry-type risk signals cause further deductions

## Decision Logic

| Score | Status | Max Loan | Rate | Tenor |
|-------|--------|----------|------|-------|
| 75–100 | Approved | 100% | 10% p.a. | 60 months |
| 60–74 | Approved | 80% | 12% p.a. | 48 months |
| 50–59 | Approved | 60% | 14% p.a. | 36 months |
| 40–49 | Conditional | 40% | 16% p.a. | 24 months |
| < 40 | **Rejected** | — | — | — |

**Hard rejection:** Any severity-5 (critical) risk signal triggers automatic rejection regardless of score, with the specific signal cited in the reason.

## CAM Document

The generated CAM includes:
1. **Executive Summary** — AI-generated overview of the credit proposal
2. **Applicant Profile** — Company background and promoter info
3. **Five Cs Analysis** — Each C with narrative + scoring breakdown
4. **Financial Analysis** — Metrics table from ingestion data
5. **Risk Matrix** — Identified risks with evidence
6. **Credit Score Summary** — Tabulated scorecard
7. **Recommendation & Decision** — Final verdict with full reasoning

Narrative sections are generated by Gemini 2.5 Flash. The document is built using `python-docx` and converted to PDF via `docx2pdf`.
